# 腾讯宣布企业微信正式开源CLI AI可调用日程、文档等7大能力

据“微信公开课”公众号介绍,此次开源优先面向10人及以下规模的企业,覆盖日常办公中最核心的协同场景: 消息与通讯录:支持获取单聊/群聊消息、发送消息,以及读取成员信息。 文档与智能表格:支持新建、读取及写入文档与智能表格。 日程与会议:支持创建、查询、更新日程,查看闲忙状态,以及预定、取消会议。 待办任务:支持创建待办、更新状态及添加参与人。 这意味着开发者可以快速构建AI应用,实现一句话创建会议、自动汇总消息等复杂任务。 想体验或开发的用户,3步即可接入体验: 配置:在企业微信后台创建机器人,获取 Bot ID 和 Secret 进行配置。 安装:安装CLI(npm install -g @wecom/cli),安装CLI SKILL(npx skills add wecom/cli -y -g) 调用:利用项目提供的 skills(技能),快速调用消息、文档等能力。
