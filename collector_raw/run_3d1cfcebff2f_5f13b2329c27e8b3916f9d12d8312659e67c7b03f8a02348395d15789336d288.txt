# 概述

钉钉客联已经暂停新客户支持,进入EOL(end of life)阶段,现有客户可继续使用已开通的功能,敬请期待新的开放能力支持,更多信息可随时关注服务端 API 更新日志。 钉钉客联是钉钉推出的跨组织沟通PaaS解决方案,专为企业构建内外部统一沟通平台而设计,实现钉钉组织内外用户间的无缝消息互通与业务协同。 钉钉客联打通了钉内账号和钉外账号的消息互通桥梁,构建了企业对内办公(2B)和对外服务/连接(2C)的统一的沟通平台,通过钉钉实现让企业员工不再频繁切换各种沟通工具完成沟通。 钉钉客联解决了企业内外部沟通割裂的痛点,通过技术赋能实现: 打破沟通壁垒 消除组织边界,建立企业员工与外部用户间的直接沟通渠道。 提升服务效率 减少工具切换,将沟通深度融入业务流程,提高响应速度。 提供标准化API,缩短系统对接周期。 跨平台消息一致性,确保用户在不同终端获得相同体验。 1. 业务系统无缝集成 提供API服务体系,支持快速对接客服、物流、招聘等核心业务系统。 开放消息收发、群管理、用户关联等关键能力,实现业务与沟通的深度融合。 支持生态伙伴基于开放能力开发行业定制化解决方案。 2. 跨平台消息协同 一套接口适配多终端(APP、小程序、网页、钉钉)。 消息实时同步,确保跨平台沟通体验一致性。 完整消息留痕与数据沉淀,支持后续分析与服务优化。 3. 场景化服务构建 支持构建客户服务、用户运营、协作沟通等标准化场景。 提供二级会话管理,将分散的外部沟通聚合为统一服务入口。 灵活的渠道配置能力,满足不同业务场景的个性化需求。 4. 互通群闭环管理 完整的钉钉客联互通群生命周期管理(创建、成员管理、信息修改、解散等)。 群模板化配置,快速部署标准化群组服务。 支持钉钉客联普通群与钉钉客联两人群两种互通群类型,满足多样沟通需求。 钉钉客联普通群是正常的群聊,钉钉客联两人群是钉钉客联基于单聊场景特殊实现,钉钉客联普通群与钉钉客联两人群对比如下:
