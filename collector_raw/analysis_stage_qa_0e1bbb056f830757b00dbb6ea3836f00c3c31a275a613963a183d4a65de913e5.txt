## Notion AI becomes an “agent” instead of an assistant

The first thing announced was the update to **Notion 3.0** .

Starting with this release, Notion AI has evolved significantly, becoming an " **AI agent** ." This marks another step in the evolution of Notion.

![Image 3](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.04.10_2x_e2656a20-34ab-4630-a039-48d75a40359f.png?v=1760441292)

Until now, Notion AI has mainly been used to edit and summarize text, acting as an "assistant" to assist with tasks.

In this evolution to "AI Agent"

*   **Search external information such as Slack, calendars, and email apps**
*   **Gather information and organize it into pages**
*   **Create a database to manage it**

**AI has been empowered to become an agent that can take over tasks** such as:

Tasks that have previously been done manually will now be performed by AI teammates who will do them for you.

![Image 4](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.13.39_2x_e11af400-5f24-43c6-b264-c5f0e4fa6633.png?v=1760441292)

From here, we will introduce **what you can specifically do** with the more powerful Notion AI.

* * *

## "Notion AI Personalization" with memory function

The first feature announced was the "AI personalization" feature.

A feature similar to "memory" in ChatGPT has been added, allowing you to instruct Notion AI on how it should behave at all times and specify which pages and databases it should reference!

![Image 5](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.20.19_2x_5643abac-ba88-4816-a13c-1b3e8decd855.png?v=1760441293)

Instructions for the AI ​​can be created on a Notion page, so you can mention pages within Notion, embed databases, and pass all of the information you've stored in Notion to Notion AI.

This means that, for example, when writing a diary, you can have it refer to past diary entries to create a summary, or read a work minutes page and automatically create a task.

![Image 6](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.28.24_2x_fa66efd7-41f8-47a3-94c5-192cff77554b.png?v=1760441292)

* * *

## Notion AI-powered database creation and editing has been further improved

Notion AI creates a database for you with its automatic database creation feature.

This time, it has been significantly improved and now supports converting databases from existing pages and editing databases!

![Image 7](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/image_542a4a3b-c978-4352-8b29-a4707eb15d68.png?v=1760441292)

In addition to creating new databases, you can also give instructions such as "make this page a database" or "change the properties like this."

You can also have them create and edit a database by pulling information from external tools such as Slack and Notion Mail.

![Image 8](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.14.31_2x_a12ad832-ca18-434b-a575-ffa532958b36.png?v=1760441292)

Notion AI's database automation is also explained in[**this**](https://kurashi-notion.com/blogs/notion/notionai-database2) article, so be sure to check it out!

* * *

## Introducing "AI Formula," which automatically writes functions!

And what's great is the introduction of "AI Formula," which allows Notion AI to automatically write functions.

I think there are quite a few people who are not good at using Notion's functions.

This was particularly difficult with complex functions, but now you can simply give instructions to the AI ​​and it will write the functions automatically!

![Image 9](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.31.29_2x_2d28fa03-1af0-4af8-9dd1-6dd206aadad1.png?v=1760441292)

For example, it can create complex conditional expressions such as "Show the number of days remaining in the project" or "If this property is...".

![Image 10](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.09.12_2x_828ae97a-c99b-4cca-8894-a7a2535f9716.png?v=1760441292)

* * *

## Introducing the ninth view, "Map View"!

Additionally, a ninth new view has been added to the database: the Map view!

Map view allows you to enter location information into the database, and you can view all registered pages on a map at once.

![Image 11](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.16.04_2x_54a7f985-cedb-4c9a-8326-8694ada9180a.png?v=1760441293)

I'm sure many people use Notion to plan their trips, so this is a very welcome update.

If you enter the name of a location, such as "Tokyo Tower," the registered location will be displayed on the map and you can open the page from there.

When it was announced at Make with Notion, it was said to be coming soon, but it's already available, so be sure to give it a try!

* * *

## Permissions can now be managed in user properties

One of the challenges when using Notion as a team is managing permissions.

A new feature has been announced that makes it easier to edit permissions!

Previously, permissions could only be edited for one page or database at a time, but now it is possible to edit permissions on a page-by-page basis within a database.

![Image 12](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.29.21_2x_46b86a8f-0214-49a3-bbe6-9b4c8ce43f87.png?v=1760441292)

Specifically, you can assign users using user properties in the database and set permissions for those users.

This means you no longer have to go into the page and change the sharing settings.

![Image 13](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.32.28_2x_1c681d4d-f67b-4d5c-9d9d-7cbd2be5054a.png?v=1760441292)

You can switch page permissions the moment you assign them, which can also be combined with automatic user addition via automation, making permission management even more convenient.

It is now also possible to set different permissions for each page in a single database, which is convenient when you want to show a page only to participants, such as for meeting minutes.

* * *

## Connecting Notion AI to other tools with MPC

Next up was an update to Notion's MCP.

MCP is a standard for connecting Notion AI with other AI tools. For example, it is now possible to easily connect editor tools such as Cursor to Notion using MCP.

![Image 14](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.10.51_2x_bec7831f-e75a-499f-9008-ce2cb3d2b5b8.png?v=1760441292)

This also strengthens integration with Notion Calendar and Notion Mail, allowing you to pull information and execute tasks across apps, such as having Notion AI read your schedule and send you an email.

Currently, there are very few AIs that can search for external information such as schedules, emails, Slack, Google Drive, etc. and use it as triggers or take action, so this is a very valuable update.

I want to actively use Notion Calendar and Notion Mail.

* * *

## Automate AI with custom agents

![Image 15](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.18.18_2x_696d3915-f049-4caf-9ece-4a9ecb324291.png?v=1760441293)

Finally, Ivan announced Notion AI's " **Custom Agent** " feature in a slide titled "One more thing." This new feature is like **AI automation** , allowing you to specify triggers for Notion AI at specific times or with specific actions (e.g., "when a Slack message arrives") and then instruct it on subsequent actions!

![Image 16](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.38.48_2x_3cd142f7-c374-4b55-899c-f7fa442f07c4.png?v=1760441292)

This makes it possible to automate things like having Notion AI automatically answer questions when they come in on Slack, or having Notion AI add a task to Notion when you ask it to "make this a task" on Slack.

Furthermore, automations created with this custom agent can be shared with others or published in the template store, so you can use automations created by others.

For example, if you use Notion as a team, everyone can use a recipe created by someone.

![Image 17](https://cdn.shopify.com/s/files/1/0743/4829/8516/files/CleanShot_2025-10-10_at_19.05.32_2x_3dd1d5f0-e4ec-48a2-8bd1-00214cdd1f1c.png?v=1760441293)

The custom agent feature will be available soon, so be sure to sign up for the waitlist [**here**](https://notion.notion.site/26fefdeead05803ca7a6cd2cdd7d112f) .
