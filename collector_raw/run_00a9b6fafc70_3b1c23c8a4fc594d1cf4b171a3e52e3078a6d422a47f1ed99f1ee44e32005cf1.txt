云文档的导出需要使用三个接口。

1.

【[创建导出任务](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/drive-v1/export_task/create)】传递指定参数创建导出任务获得导出任务id

2.

【[查询导出任务结果](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/drive-v1/export_task/get)】根据导出任务id与导出文档 token 获取导出结果

3.

【[下载导出文件](https://open.feishu.cn/document/uAjLw4CM/ukTMukTMukTM/reference/drive-v1/export_task/download)】根据导出任务结果中获取到导出文件 token 下载对应的文件

## 云文档支持导出的类型[#](https://feishu.apifox.cn/doc-1950023#%E4%BA%91%E6%96%87%E6%A1%A3%E6%94%AF%E6%8C%81%E5%AF%BC%E5%87%BA%E7%9A%84%E7%B1%BB%E5%9E%8B)

导出文件的扩展类型，必须和云文档的类型相匹配 doc(飞书云文档): 支持导出为 Microsoft Word (DOCX)、pdf docx(新版飞书云文档): 支持导出为 Microsoft Word (DOCX)、pdf sheet(飞书电子表格): 支持导出为 Microsoft Excel (XLSX)、csv bitable(飞书多维表格): 支持导出为 Microsoft Excel (XLSX)、csv
## 导出文件存储时间[#](https://feishu.apifox.cn/doc-1950023#%E5%AF%BC%E5%87%BA%E6%96%87%E4%BB%B6%E5%AD%98%E5%82%A8%E6%97%B6%E9%97%B4)

导出的文件会在任务结束10分钟后删除，删除后无法下载。
## 导出文档限制[#](https://feishu.apifox.cn/doc-1950023#%E5%AF%BC%E5%87%BA%E6%96%87%E6%A1%A3%E9%99%90%E5%88%B6)

导出 Microsoft Word (DOCX) 格式限制文档资源最大 1GB 导出 pdf 格式限制文档资源最大 128MB
## 知识库文档导出[#](https://feishu.apifox.cn/doc-1950023#%E7%9F%A5%E8%AF%86%E5%BA%93%E6%96%87%E6%A1%A3%E5%AF%BC%E5%87%BA)

知识库文档导出需要先通过[获取节点信息](https://open.feishu.cn/document/ukTMukTMukTM/uUDN04SN0QjL1QDN/wiki-v2/space/get_node)得到文档的真实 token 与文档类型，然后通过**云文档导出流程**进行导出。
## 水印[#](https://feishu.apifox.cn/doc-1950023#%E6%B0%B4%E5%8D%B0)
