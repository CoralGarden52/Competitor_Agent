# 概述- 钉钉开放平台

本文档介绍了MCP 服务和MCP 广场是什么、能力概览及如何接入MCP服务,帮助开发者能快速实现能力对接。 MCP(Model Context Protocol)是一种给大模型使用的连接协议,但对企业用户而言,真正使用的是基于该协议封装好的 MCP 服务,即一个个标准化、可被大模型直接调用的“技能”。 你可以把每个 MCP 服务理解为: 一个预集成好的工具包(比如“合同风险审核”“会议语音转写”“发票信息抽取”); 一个安全可控的能力单元,企业只需授权,大模型就能在合规范围内调用; 一种无需开发即可接入的 AI 能力,直接用于 AI Agent、自动化工作流或智能应用中。 目前,钉钉已提供 6000+ 企业级 MCP 服务,是中国规模最大的企业级 AI 技能市场。 钉钉6000+ MCP 服务来自两个核心供给方,覆盖从基础办公到专业行业的全场景需求: 钉钉官方产品能力封装 钉钉将自身高频使用的办公协同能力,直接转化为 AI 可调用的服务,通常不额外消耗企业算粒:钉钉文档 AI、AI 表格、钉钉会议、日程、待办、审批等流程操作等,这些构成了钉钉企业 AI 应用的基础能力底座。 生态能力汇聚于“钉钉 MCP 广场” MCP 服务广场是钉钉“AI 能力中台”的对外服务门户,提供标准化、即插即用的 AI 能力集合。所有能力均封装为 MCP 服务,支持开发者通过低代码或 API 灵活集成,快速构建智能客服、自动化流程、行业解决方案等应用。 可点击链接访问 MCP 广场。 MCP广场重点引入高价值能力,所有能力均以标准 MCP 服务形式提供,覆盖以下关键场景: 文生图、图生视频、语音合成(TTS)、文本续写与润色 智能信息处理 高效解析非结构化数据,提升信息处理速度 降低文档处理成本,提升办公效率 能力扩展:如需新增能力需求,请提交表单:能力需求表单 优势 说明 开箱即用,集成门槛极低 无需开发团队写代码对接,只需在钉钉工作流、Agent 编排器或 Deap 技能中心中选择服务、简单配置,即可启用。
